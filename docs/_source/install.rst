Installation
===============

----------------

**Dashboard & Visualization**

======================= ===================  ===================

======================= ===================  ===================

networkx                
======================= ===================  ===================

**Basic Computing**

======================= ===================  ===================

**Miscellaneous**

======================= ===================  ===================

Installation
---------------------

You can install the package by pip:

.. code-block::

    pip install mocharts -U
